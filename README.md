## 📌 What is Dr. Well?

Dr. Well is an **AI-powered Medical Assistant** that provides instant healthcare guidance, symptom analysis, medication recommendations, and doctor referrals.

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI Consultation** | 24/7 symptom analysis & instant diagnosis |
| 💊 **Smart Medications** | Auto-prescription with food restrictions |
| 👨‍⚕️ **Doctor Database** | 15+ verified specialists with complete details |
| 📅 **Appointment Booking** | Easy scheduling with doctors |
| 🍎 **Nutrition Advice** | Condition-based dietary guidance |
| 👤 **User Profiles** | Separate portals for patients & doctors |

## 🚀 Quick Start

# Clone & Install
git clone https://github.com/yourusername/dr-well.git
cd dr-well
pip install -r requirements.txt

# Set API Key
echo "GOOGLE_API_KEY=your_key_here" > .env

# Run App
streamlit run main.py

## 🎯 What Dr. Well Can Do

- ✅ Analyze symptoms (fever, headache, cold, cough, chest pain)
- ✅ Prescribe medications with food tips
- ✅ Detect emergencies (chest pain → cardiologist)
- ✅ Show complete doctor details (name, clinic, fee, timings, phone)
- ✅ Book appointments
- ✅ Track medications & appointments
